# Credit Risk Prediction Project
This project was completed as part of my **Data Science Internship at Developers Hub.

# Project Overview
The goal is to predict whether a loan should be approved based on applicant details like income, education, and credit history using **Logistic Regression**.

# Key Results
* Accuracy: 78.86%
* Cleaning: Handled missing values using Mode/Median imputation.
* Visualization: Included analysis of Income, Education, and Loan Amounts.

# Files in this Repo
* `Credit_Risk_Analysis.ipynb`: Full code and data visualizations.
* `credit_risk_model.pkl`: The trained model file (ready for deployment).
