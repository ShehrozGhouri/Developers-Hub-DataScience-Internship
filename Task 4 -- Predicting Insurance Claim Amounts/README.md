# Task 4: Medical Insurance Claim Prediction
This project involves building a Machine Learning model to predict medical insurance claim amounts based on individual personal data.

### 🎯 Objective
* Preprocess and clean the Medical Cost Personal dataset.
* Encode categorical data and analyze feature correlations.
* Train a regression model to predict insurance charges.

### 🧠 Model Details
* **Algorithm:** Linear Regression
* **Accuracy Score:** 78.36% (R-squared)

### 📁 Folder Contents
* `Insurance_Claim_Prediction.ipynb`: Full code and visualization.
* `insurance.csv`: The dataset used for training.
* **Model/**: Contains the saved `.pkl` file for deployment.
